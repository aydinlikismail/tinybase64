# ***Tiny Base64***
```js
const base64 = require('tinybase64');

const encodedtext = base64.encode('Hello!');
console.log(encodedtext) // Result: SGVsbG8h

const decodedtext = base64.decode('SGVsbG8h');
console.log(decodedtext) // Result: Hello!
```
